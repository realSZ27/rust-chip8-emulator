Extremely simple two file [CHIP-8](https://en.wikipedia.org/wiki/CHIP-8) emulator (interpreter) written in Rust.

I wrote this pretty much just to learn about emulator basics and Rust. It's not intended to be useful in any way, though it may be nice as a refrence. It passes all flags and corax+ opcode tests from [here](https://github.com/Timendus/chip8-test-suite), but fails 3/6 of the quirks tests.
